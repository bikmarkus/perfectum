Принцип работы:
Открываем главную страницу perfectum/
На ней размещаем view для Корзины и ссылки на остальные три модуля.
При переходе по ссылке вызываем через AJAX контроллер модуля и передаем ему нужные параметры в $_POST.
Контроллер вызывает нужные методы модели и отдает их view.
View отображает нужные данные в исходном блоке.
Название контроллера определяем по переменной $_SERVER['REQUEST_URI'].
